##  Pet Projects Platform

A platform for developers who want real APIs and clean UI templates to practice with.

It provides:

- Free REST & GraphQL APIs
- API keys and rate limits
- Sorting, pagination, and configurable limits
- Curated Figma UI templates

The goal is to provide realistic infrastructure so developers can focus on features instead of backend boilerplate.
